# Research Roadmap

## Phase 1: Foundation

- Establish normalized event models.
- Support reproducible local CSV ingestion.
- Add baseline repetition and co-occurrence detectors.
- Generate Markdown analyst summaries.
- Maintain CI with linting, type checks, and tests.

## Phase 2: Graph analysis

- Build actor-artifact-event graphs with NetworkX.
- Add centrality and community-detection summaries.
- Export graph metrics for reports.

## Phase 3: Anomaly detection

- Add time-windowed event burst detection.
- Add similarity clustering for repeated narratives.
- Add confidence scoring that separates weak signals from stronger multi-factor indicators.

## Phase 4: Analyst workflow

- Add case folders and structured report templates.
- Add evidence manifests with hashes and provenance notes.
- Add privacy-preserving redaction helpers.
