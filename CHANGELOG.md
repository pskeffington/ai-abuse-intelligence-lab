# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - Unreleased

### Added

- Typed domain models for sources, actors, artifacts, normalized events, and findings.
- CSV ingestion for reproducible local analysis fixtures.
- Repeated-signal analysis for tags and actor handles.
- Rolling-window timing analysis for concentrated event activity.
- Actor-artifact-event graph construction with compact graph metrics.
- Combined baseline analysis pipeline.
- Markdown and JSON finding reporters.
- Typer-based command line interface.
- Synthetic sample event dataset.
- Unit tests, strict mypy configuration, ruff linting, and GitHub Actions CI.
- Governance and publication-readiness documentation.
